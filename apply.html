How To Apply

Below are the next available course dates.

To apply you will need to download an application form, fill it in and then return it to Mrs Aruna Krishna (full details on application form). To download the form right click on either the Word format (if you have Microsoft Word) or the PDF format (if you prefer to use Acrobat Reader), select "save target as" and when prompted, save it to disk. When the download is complete open the file and print it out.
If applying within 2 weeks of start date of the course, please contact us by phone or e-mail to check availability of places.
